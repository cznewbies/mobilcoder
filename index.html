    MobilCoder is free application that can edit HTML5, CSS3 and scripting language and view
						it, all on mobile.

						 Tomáš Wróbel. All rights reserved
    
				
    
				
    
					
    Scripting language can be JavaScript, TypeScript and React
    
					
    JavaScript var newLine = document.createElemnt("br");
					
    
					
    TypeScript var newLine: HTMLBRElement = document.createElemnt("br");
